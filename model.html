Some further thoughts to the model:


  1. Email has been added to user.

  2. With regards to stories and comments there are some new issues. It seems that on most sites comments can be commented as well so probably merging the two tables is the easiest way to deal with it. As a result the previous STORY table has changed and the COMMENTS table has been deleted.

  3. Since all comments and stories get a unique identifier from the same table one LIKES table is enough.
